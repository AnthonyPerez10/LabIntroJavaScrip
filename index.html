Introducción

El uso de javascript se utiliza para el desarrollo de páginas web más interactivas para el usuario, donde se puede agregar un diseño más moderno y esencial a las páginas web, en este documento se hablará sobre los principales conceptos de JavaScript como se puede mencionar Tipos de datos, operadores, clases, estructuras de control, funciones entre otros.
Donde se mostrarán ejemplos y teoría de cada uno de ellos.

a. Introducción a JavaScript

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, ampliamente utilizado para crear interactividad en páginas web. Permite manipular el DOM (Document Object Model), manejar eventos y realizar operaciones del lado del cliente. Es esencial para el desarrollo web moderno, complementando HTML y CSS.

i. Conceptos de programación

La programación en JavaScript se basa en conceptos fundamentales como variables (para almacenar datos), funciones (bloques de código reutilizables), objetos (estructuras que agrupan propiedades y métodos), y el paradigma de programación funcional (como el uso de funciones de orden superior). JavaScript es dinámico, lo que significa que las variables no necesitan un tipo fijo, y soporta tanto programación imperativa como funcional.

b. Sintaxis

La sintaxis de JavaScript es flexible y similar a lenguajes como C o Java. Incluye declaraciones, expresiones, y el uso de llaves `{}` para bloques de código. Es sensible a mayúsculas y minúsculas, y las instrucciones terminan con punto y coma (`;`), aunque no siempre es obligatorio.

i. Tipos de datos

JavaScript tiene tipos de datos primitivos y complejos. Los primitivos incluyen números, cadenas, booleanos, etc., mientras que los complejos son objetos y arreglos.

1. Numbers

Representan valores numéricos, enteros o flotantes. JavaScript no distingue entre enteros y flotantes.

                let edad = 25; // Entero
                let precio = 19.99; // Flotante
                console.log(edad + precio); // 44.99

2. Strings

Cadenas de texto, delimitadas por comillas simples o dobles.

                let nombre = "Ana";
                let saludo = 'Hola, ' + nombre;
                console.log(saludo); // "Hola, Ana"

3. Arrays

Colecciones ordenadas de elementos, que pueden ser de cualquier tipo.

                let frutas = ["manzana", "banana", "naranja"];
                console.log(frutas[1]); // "banana"
                frutas.push("pera"); // Agrega un elemento

4. Objects (JSON, Object Destructure, Spread Operator, etc)

Objetos son colecciones de pares clave-valor. JSON es un formato de texto para representar objetos. La destructuración permite extraer propiedades fácilmente, y el operador spread (`...`) copia o fusiona objetos.

            // Objeto básico
            let persona = { nombre: "Juan", edad: 30 };
            
            // JSON
            let jsonPersona = JSON.stringify(persona); // Convierte a string JSON
            console.log(jsonPersona); // '{"nombre":"Juan","edad":30}'
            
            // Destructuración
            let { nombre, edad } = persona;
            console.log(nombre); // "Juan"
            
            // Spread Operator
            let copiaPersona = { ...persona, ciudad: "Madrid" };
            console.log(copiaPersona); // { nombre: "Juan", edad: 30, ciudad: "Madrid" }

ii. Operadores

Operadores realizan operaciones en valores. Incluyen aritméticos (`+`, `-`, `*`, `/`), de comparación (`==`, `===`, `>`, `<`), lógicos (`&&`, `||`, `!`), y de asignación (`=`, `+=`).

                let a = 5;
                let b = 10;
                console.log(a + b); // 15 (aritmético)
                console.log(a === 5); // true (comparación estricta)
                console.log(a > 3 && b < 20); // true (lógico)

iii. Estructuras de control

Controlan el flujo del programa: condicionales (`if-else`), bucles (`for`, `while`), y switches.

                // Condicional
                let numero = 10;
                if (numero > 5) {
                    console.log("Mayor que 5");
                } else {
                    console.log("Menor o igual a 5");
                }
                
                // Bucle for
                for (let i = 0; i < 3; i++) {
                    console.log(i); // 0, 1, 2
                }
                
                // Switch
                let dia = "lunes";
                switch (dia) {
                    case "lunes":
                        console.log("Inicio de semana");
                        break;
                    default:
                        console.log("Otro día");
                }

IV. Funciones: map(), reduce(), filter()

Estas funciones permiten manipular arreglos de forma funcional:

  • map(): Crea un nuevo arreglo aplicando una función a cada elemento.
  • filter(): Filtra los elementos que cumplan una condición.
  • reduce(): Reduce los elementos a un solo valor (por ejemplo, suma total).
            // Ejemplo:
            const numeros = [1, 2, 3, 4, 5];

            const cuadrados = numeros.map(n => n * n);
            const pares = numeros.filter(n => n % 2 === 0);
            const suma = numeros.reduce((total, n) => total + n, 0);

            console.log(cuadrados); // [1, 4, 9, 16, 25]
            console.log(pares);     // [2, 4]
            console.log(suma);      // 15

V. Clases en JavaScript

Las clases son una forma moderna de definir objetos con propiedades y métodos.

            class Persona {
            constructor(nombre, edad) {
                this.nombre = nombre;
                this.edad = edad;
            }

            saludar() {
                return `Hola, soy ${this.nombre} y tengo ${this.edad} años.`;
            }
            }

            const persona1 = new Persona("Ana", 25);
            console.log(persona1.saludar());

VI. Hoisting de Variables

            console.log(x); //undefined, porque se eleva la declaración pero no el valor
            var x = 10;

            saludar(); // Funciona por hoisting de funciones

Conclusión

El uso de javascript es una forma de darle mejorar forma a las paginas web para un mejor diseño y una mejor apariencia visual, además, los conceptos de JavaScript como las funciones de orden superior (map, filter, reduce), las clases, el hoisting, la manipulación del DOM y los eventos constituyen la base del desarrollo web moderno. Las funciones permiten trabajar con arreglos de manera más eficiente y declarativa, las clases facilitan la creación de objetos y la organización del código, y el hoisting explica cómo JavaScript interpreta las variables y funciones antes de ejecutar el programa. Además, la manipulación del DOM y el uso de eventos hacen posible crear páginas interactivas, dinámicas y con respuesta inmediata a las acciones del usuario.
